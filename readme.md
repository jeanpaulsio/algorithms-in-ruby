A collection of classic computer science algorithms written in Ruby

## Getting Started

```bash
$ git clone https://github.com/jeanpaulsio/algorithms_in_ruby.git
$ bundle install
```

## Running Tests

```bash
$ ruby lib/arrays_and_strings/reverse_string_test.rb
```

## Getting Started

```bash
$ git clone https://github.com/jeanpaulsio/algorithms_in_ruby.git
$ bundle install
$ asdf install
```

## Running Tests

```bash
$ ruby lib/arrays_and_strings/reverse_string_test.rb
```

## Watching Tests

If you prefer not to run tests manually, you can run:

```
$ bundle exec guard
```

## Prettier-ing files

```bash
bundle exec rbprettier --write '**/*.rb'
```
